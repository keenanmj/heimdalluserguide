Quick Start

Last updated: January 29th, 2020

Account Creation

Before you can begin your Heimdall experience, you must first create an account to obtain the necessary materials to get Heimdall up and running.


To create an account first visit the Heimdall website. You will be prompted to input a username and password. However, if you are looking to create a new account you will not have these credentials just yet. On this initial page you will see a create account button to click for new users. Click this button and you will be prompted to fill out a simple form to get an account created. Once you’ve filled out the form and click the create account button located at the bottom of this form, congratulations! You now can get access to Heimdall!

Download

The necessary materials to ensure a successful Heimdall installation can be first downloaded from the Heimdall site. First, you must create a user account to be able to obtain these materials. If you have not already created an account, please refer to the Account Creation section on this page.


After account creation, navigate to the login page of the Heimdall website. Enter your user credentials and click the login button. You will be redirected to the splash page of the Heimdall website. Once there, click the tab labeled ‘Agent’ to move to the Agent page. From here, you will be able to download Heimdall’s materials. To initiate the download, click the add agent button located near the middle of the webpage. This will begin downloading a zipped tar folder entitled Heimdall_v_xxxxxx.tar.gz that contains all the necessary files for agent installation on the target server. Once this folder is downloaded, you can now move on to the installation phase.

Installation

Once the download completes, you should extract the folder to the /usr/local directory on the system. Navigate to the directory where the downloaded folder is stored and extract it with the following command:

tar -C /usr/local -xzf Heimdal_v_xxxxxx.tar.gz

Once this command completes, navigate to the /usr/local folder and then move into the newly extracted Heimdall folder. Inside of the Heimdall folder, there are the necessary executable and binary files associated with Heimdall. Inside of this directory will be a read me file. This read me contains detailed instructions to follow as a guided installation process. Please consult this file for the finishing installation steps to set up the agent on the target server.

You should now be setup and ready to monitor. May Heimdall watch over you!
