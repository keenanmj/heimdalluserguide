Quick Start

Last updated: January 29th, 2020

Account Creation

Before you can begin your Heimdall experience, you must first create an account to obtain the necessary materials to get Heimdall up and running.


To create an account first visit the Heimdall website. You will be prompted to input a username and password. However, if you are looking to create a new account you will not have these credentials just yet. On this initial page you will see a create account button to click for new users. Click this button and you will be prompted to fill out a simple form to get an account created. Once youâ€™ve filled out the form and click the create account button located at the bottom of this form, congratulations! You now can get access to Heimdall!

Installation

Installation is simple and only requires NodeJS and PostgreSQL.

Install the server requirements:

cd SeniorDesign/server

npm install .

Start the server:

npm start

Navigate the web server url (default: http://localhost:5432) in your browser and follow the on-screen insructions to create a user account.

Configuration

On-screen instructions will walk you through configuring the web server, new users, and new monitoring agents.

Useful Tip:

An agent configuration file will be automatically generated when creating a new connection.
